Display your latest twitter feed in your page, post or text widget using shortcode.No API required.


== Description ==

The Page Plugin lets you easily embed and promote any Twitter feed on your website. Just like on Twtitter,


There is no need to create a Twitter application and provides a nice interface to display your tweets in an easy way, just install the plugin and set the options.
your visitors can like and share the Page without having to leave your site. 

[tfws username="your user name"] use shortcode in page or post .

<?php echo do_shortcode("[tfws username='your user name']") ?>


= Features =
1. Very Easy to setup and use.
2. Don't need to create an API.
2. Just install the plugin, and use the shortcode in page,post and wordress default text widget.
3. Comes with shorcode [tfws username="swadeshswain" theme="dark" color="#FAB81E" tweets="2" header="no" footer="no" borders="yes" scrollbar="yes" background="no"]

== Installation ==
Installing this plugin is pretty simple like all other wordpress plugins. Here are 2 ways to install :

<h3>How  to  Install</h3>

1. Download the zipped plugin from wordpress repository
2. Extract the zipped folder
3. Upload twitter-feeds-with-shortcode to the /wp-content/plugins/ directory
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Go To Widgets page and drag Facebook Fan Page Widget to an Widget area.
6. Select the options and Save




== Frequently Asked Questions == 
= How can I Show it on my Website ? =
1. Go To  page and post click the twitter icon on tinymice editor to generate the shortcode or you can copy this shortcode [tfws username="swadeshswain" theme="dark" color="#FAB81E" tweets="2" header="no" footer="no" borders="yes" scrollbar="yes" background="no"] and past it on your page, post or widget 
2. Change the option as per your requirement and then save the page or post.
3. The twitter feed will be showed on your website!
